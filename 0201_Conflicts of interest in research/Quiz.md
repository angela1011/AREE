# 課後測驗

1. 當研究者或研究機構對於主要利益的專業判斷，受到次要利益的不當影響時，常會產生利益衝突。請問下列何者不是研究上常見的次要利益？
```bash
知識
```

2. 研究機構的利益衝突與個人的利益衝突狀況相仿，當發生利益衝突時，研究機構須考量到其主要任務。請問下列何者不是研究機構的主要任務？
```bash
經濟利益
```

3. 下列何者不是管理利益衝突的重要性？
```bash
確保研究的經濟利益
```

4. 下列何者不是利益衝突的處置原則？
```bash
利益交換
```

5. 為避免研究倫理委員會的成員陷入利益衝突的困境，下列何者可能最不適任被指派參與審查研究計畫？
```bash
研究計畫申請人的老師
```
